# AGENTS.md (/apps-script)

## Overview
Google Apps Script source files for Google Workspace Drive automation.

## Requirements
- Node.js 16+
- Google Cloud project with Drive API enabled
- Google Workspace account

## Setup

### Initial Setup
```bash
npm install -g @google/clasp
clasp login
```

### Clone Existing Script
```bash
cd apps-script
clasp clone <YOUR_SCRIPT_ID>
npm install
```

See `package.json` in this directory for npm dependencies.
